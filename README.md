# About Me

**Backend & Systems Developer | Freelancer | 42 Madrid**

Empecé a programar de forma autodidacta siendo un niño, y ese mismo impulso es el que me sigue moviendo hoy. Desarrollo soluciones completas como freelancer para particulares y pequeños equipos: desde la arquitectura backend hasta el frontend, con plataformas que han alcanzado más de 100.000 usuarios.

Common Core completado en 42 Madrid (tasa de finalización: 8%). Actualmente cursando el máster en Arquitectura de Datos (RNCP 7). Interés fuerte en LLMs, despliegue de modelos en local y todo lo que tenga que ver con entender cómo funcionan las cosas por dentro.

## Lo que hago

- Sistemas backend pensados para rendimiento, seguridad y escalabilidad — C, C++, Go
- Desarrollo full stack con enfoque backend-first — TypeScript, Vue, React
- Diseño de bases de datos y arquitecturas de datos — PostgreSQL, MariaDB, MongoDB, Python
- Despliegue y mantenimiento de aplicaciones en producción con carga real
